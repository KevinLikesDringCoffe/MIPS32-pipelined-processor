# MIPS32-pipelined-processor
A MIPS32 pipelined processor impelmented by Verilog HDL
## Version 1.0.200701
### Current functional unit
- Program counter
- Instruction ROM
- Instruction decoder
- Harzard resolve unit(partly)
- Execution unit
- Register file
